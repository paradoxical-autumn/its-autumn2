ACA

C#

Resonite

ACA or Arbitrarily Contained Anarchy (formerly Arbitrary Component Access) is a Resonite Plugin designed to extend FrooxEngine's functionality by adding in some more "dangerous" features. The first features I and my partner for the project - ErrorJan - were for Arbitrary Component Access: grabbing fields and values off of components you're not supposed to access. The plugin would later expand into general access to dangerous functionality such as automated spawning. The project is always in development and still sees updates to this day.
